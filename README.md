StayBuddy 🏡

Project Overview ✨

StayBuddy is a personal web application developed using Node.js, Express, and MongoDB (Mongoose ODM). This project showcases a robust system for managing accommodation listings, complete with user authentication, secure image uploads via Cloudinary, and comprehensive CRUD (Create, Read, Update, Delete) functionalities for listings. It represents a practical application of modern web development technologies, demonstrating skills in full-stack development.

Key Features 🚀

•User Authentication: Secure signup, login, and logout processes, ensuring data privacy and user access control.
•Accommodation Management: Full CRUD operations for creating, viewing, updating, and deleting accommodation listings, providing complete control over content.
•Image Uploads: Seamless integration with Cloudinary for efficient and scalable image storage for listings, enhancing visual appeal and performance.
•User Feedback: Intuitive flash messages provide real-time feedback for user actions, improving the overall user experience.
•Session Management: Persistent user sessions ensure a smooth and uninterrupted experience across interactions.

Technologies Utilized 🛠️

•Backend: Node.js, Express.js - Powering the server-side logic and API.
•Database: MongoDB (managed with Mongoose ODM) - A NoSQL database for flexible and scalable data storage.
•Templating: EJS (Embedded JavaScript) - Used for dynamic content rendering, creating interactive and responsive user interfaces.
•Authentication: Passport.js, Passport-Local, and Passport-Local-Mongoose - Implementing robust user authentication strategies.
•Cloud Services: Cloudinary - For cloud-based image storage, integrated using Multer and Multer-Storage-Cloudinary for efficient file handling.
•Session Handling: Express-Session and Connect-Mongo - For secure and persistent session management.
•Utilities: Dotenv for environment variable management, Joi for schema validation, Method-Override for HTTP method overriding, Connect-Flash for flash messages, and Cookie-Parser for cookie handling.

Project Functionality 💡

StayBuddy allows users to:
•Register and log in to their personal accounts securely. 🔐
•Create and manage their own accommodation listings, including adding multiple images to showcase properties. 📸
•View and interact with all available accommodation listings, exploring various options. 🗺️
•Edit or delete their previously created listings, maintaining up-to-date information. ✏️
