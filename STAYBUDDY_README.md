## 🌟 Overview

StayBuddy is a robust, personal web application designed to simplify the process of listing and discovering accommodations. It features a secure user ecosystem, dynamic property management, and cloud-integrated media handling, showcasing modern full-stack development practices.

---

## 🚀 Key Features

- **🔐 Secure Authentication**Robust signup and login system powered by **Passport.js**, ensuring user data integrity and access control.

- **🏗️ Full CRUD Functionality**Comprehensive property management allowing users to Create, Read, Update, and Delete listings with ease.

- **☁️ Cloud Image Hosting**Seamless integration with **Cloudinary** for scalable, high-performance image storage and delivery.

- **⚡ Interactive Experience**Dynamic frontend rendering with **EJS** and real-time user feedback via flash messages.

- **💾 Persistent Sessions**Reliable session management using **Connect-Mongo** for a smooth, uninterrupted user journey.

## 🛠️ Technical Stack

| Category | Tools & Technologies |
| --- | --- |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB (Mongoose ODM) |
| **Frontend** | EJS (Embedded JavaScript), CSS3, HTML5 |
| **Security** | Passport.js, Passport-Local, Passport-Local-Mongoose |
| **Cloud** | Cloudinary, Multer, Multer-Storage-Cloudinary |
| **Utilities** | Joi (Validation), Connect-Flash, Dotenv, Cookie-Parser |

---

## 📂 Project Architecture

- 📁 **`controllers/`** — Core application logic and request handling.

- 📁 **`models/`** — Mongoose schemas and data modeling.

- 📁 **`routes/`** — Express route definitions for clean URL mapping.

- 📁 **`views/`** — Dynamic EJS templates for the frontend.

- 📁 **`public/`** — Static assets including custom CSS and client-side scripts.

- 📄 **`app.js`** — Main entry point and server configuration.

- 📄 **`cloudConfig.js`** — Secure configuration for Cloudinary services.
